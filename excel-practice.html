Excel Master Class - Interactive Exercises

Select an exercise from the dropdown to practice Excel skills. Exercises progress from Beginner to Pro. Each includes downloadable data and steps for Excel or Google Sheets.

Exercise 1: SUM and AVERAGE (Beginner)

Goal: Calculate total and average expenses for a monthly budget report.

Steps:

  1. Create a table with columns: Item, Amount.
  2. Copy sample data to A1:B11.
  3. In A12, use =SUM(B2:B11) to calculate total expenses.
  4. In A13, use =AVERAGE(B2:B11) for average.
  5. Highlight Amounts > $150 in yellow: Home > Conditional Formatting > Greater Than > 150, set yellow fill.

Sample Data (A1:B11):

ItemAmount
Rent1200
Food300
Travel150
Supplies80
Utilities200
Phone100
Internet50
Books75
Gas120
Misc90
Download CSV
Tip: SUM adds all values in a range; AVERAGE finds the mean. Drag the bottom-right corner of a cell to copy formulas.

Solution

Detailed Steps:

  1. In A12, type =SUM(B2:B11). Result: $2365.
  2. In A13, type =AVERAGE(B2:B11). Result: $236.50.
  3. Select B2:B11, Home > Conditional Formatting > Greater Than, enter 150, choose yellow fill. Highlights Rent ($1200), Food ($300), Utilities ($200).

Output (Partial):

ItemAmount
Rent1200
Food300
Travel150
Supplies80
Utilities200
Total2365
Average236.50

Visual: Bar chart would show Rent, Food, and Utilities as prominent bars with a red line at $150 for high expenses.

Exercise 6: Basic VLOOKUP (Intermediate)

Goal: Lookup employee salaries for payroll processing.

Steps:

  1. Create "Employees" sheet: ID, Name, Salary (A1:C11).
  2. Create "Payroll" sheet: ID, Name, Salary (A1:C11).
  3. In Payroll C2, use =VLOOKUP(A2, Employees!A:C, 3, FALSE), drag down.
  4. Create bar chart for salaries: Insert > Bar Chart.

Exercise 8: Nested VLOOKUP (Intermediate)

Goal: Lookup department and salary from employee IDs.

Steps:

  1. Create "Employees" sheet: ID, Name, Dept ID (A1:C11).
  2. Create "Departments" sheet: Dept ID, Dept Name, Salary (A1:C11).
  3. Create "Payroll" sheet: ID, Name, Dept Name, Salary (A1:D11).
  4. In Payroll C2, use =VLOOKUP(VLOOKUP(A2, Employees!A:C, 3, FALSE), Departments!A:C, 2, FALSE).
  5. In Payroll D2, use =VLOOKUP(VLOOKUP(A2, Employees!A:C, 3, FALSE), Departments!A:C, 3, FALSE).
  6. Create bar chart for Salaries.

Sample Data (Employees A1:C11):

D1Sales50000
D2IT60000
D3HR45000
Download Employees CSV Download Departments CSV
Tip: Nested VLOOKUP uses one VLOOKUP’s result as input for another.

Solution

Detailed Steps:

  1. In Payroll C2, type =VLOOKUP(VLOOKUP(A2, Employees!A:C, 3, FALSE), Departments!A:C, 2, FALSE).
  2. In Payroll D2, type =VLOOKUP(VLOOKUP(A2, Employees!A:C, 3, FALSE), Departments!A:C, 3, FALSE). Drag to D11.
  3. Select Payroll A1:D11, Insert > Bar Chart. Add title: "Salaries by Department".

Exercise 11: Goal Seek (Pro)
